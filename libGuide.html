3D Printing

What is 3D printing?

3D printing is a prototyping/additive manufacturing technique* using 3-Dimensional computer aided drawings (CAD) files usually (.stl or .obj) to be "sliced" into specialized code (.gcode) that 3D printers can use to create physical objects. These objects are made from heated material (usually plastic), that is built up layer by layer according to the 3D model parameters within specific (X, Y, Z) axis dimensions.

(The programs are used for the physical creation of a 3D model using a 3D printer, these create .gcode files which are made up using an (X, Y, Z) geometric coordinate system specific to the model and printer bed size which are read locally on the 3D printer. It then slowly builds up the real object, click here for info on available slicing software.)
